# Concept Notes: Module 01 - Paradigm Foundations

To build high-value systems, we must classify them correctly:

## 1. Chatbots (Stateless Interaction)
- **Autonomy**: Zero.
- **Description**: Natural language interface answering questions purely from static context or input prompts.
- **Example**: Frequently Asked Questions FAQ responders.

## 2. Copilots (Interactive Context)
- **Autonomy**: Low.
- **Description**: Assists user inside a specific interface (e.g. IDE, CRM), suggesting code or email content, but relies entirely on human approval to commit state changes.
- **Example**: GitHub Copilot, cursor text autocomplete.

## 3. Autonomous Agents (State-Directed Execution)
- **Autonomy**: High.
- **Description**: Receives high-level objective, dynamically selects tools, evaluates outputs, self-corrects, and executes cycles autonomously until completion.
- **Example**: Support agent resolving refund disputes and committing database mutations autonomously.

# Setup Diagnostics

Everything you need to get your environment ready before the training starts.

## Steps

1. Read [install.md](file:///c:/Users/Richa/OneDrive/Documents/Projects/agentic-ai-zero-to-production/setup/install.md) and follow the instructions.
2. Copy [sample.env](file:///c:/Users/Richa/OneDrive/Documents/Projects/agentic-ai-zero-to-production/setup/sample.env) to `.env` in the root of the repo and fill in your API keys.
3. Run [verify_environment.py](file:///c:/Users/Richa/OneDrive/Documents/Projects/agentic-ai-zero-to-production/setup/verify_environment.py) to confirm everything is installed correctly.
4. Run [test_api_keys.py](file:///c:/Users/Richa/OneDrive/Documents/Projects/agentic-ai-zero-to-production/setup/test_api_keys.py) to confirm API access works.

If anything fails, check the troubleshooting guide in [docs/troubleshooting.md](file:///c:/Users/Richa/OneDrive/Documents/Projects/agentic-ai-zero-to-production/docs/troubleshooting.md).

## Time Required

- About 15 minutes if you already have Python and git installed.
- About 30 minutes if you are starting from scratch.

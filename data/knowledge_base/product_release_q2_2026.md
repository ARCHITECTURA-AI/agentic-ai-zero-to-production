# Product Release Notes: Q2 2026

## 🚀 Version 3.4.0 — Released May 15, 2026

### 🛠️ Key Enhancements:
1. **Model Context Protocol (MCP) Support**: Built-in support to allow custom agents to query databases and read local file trees dynamically.
2. **Hybrid In-Memory RAG Store**: ChromaDB index speeds improved by 40% with smart metadata caching enabled.
3. **Prompt Injection Firewalls**: Active filters introduced on standard completion router endpoints.

### 🐛 Bug Fixes:
- Fixed SQLite database version override errors on legacy Windows operating systems.
- Resolved memory leaks in long-running multi-agent conversation graphs.

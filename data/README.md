# Corporate Sandbox Dataset

This folder houses mock enterprise data designed for the labs, routing algorithms, RAG memory tests, and safety injections.

---

## 📂 Subfolder Index

- **`tickets/`**: Holds unstructured support ticket JSON files for the loop and multi-agent routing.
- **`knowledge_base/`**: Holds markdown policies for vector indexing inside Module 04 RAG memory.
- **`code_samples/`**: Holds buggy script executions for containment/sandbox challenges in Module 09.
- **`eval_sets/`**: Labeled golden sets to run continuous evaluation tests in Module 05.
